Features

Multi-Sheet Processing

All worksheets in your workbook are processed. Switch between sheets to review changes.

Consistency Across Sheets

Same values get the same replacement across all sheets. If john@example.com appears in Sheet1 and Sheet3, both become [EMAIL-1].

Cell-Level Precision

PII detection works at the cell level. A cell containing Contact: john@example.com becomes Contact: [EMAIL-1].

Formula Preservation

Formulas referencing cells with PII will reference the anonymized values. Calculated results update automatically.

Use Cases

  • Database exports — Sanitize customer/employee data dumps
  • Analytics reports — Remove user identifiers from behavioral data
  • Financial reports — Redact account numbers and transactions
  • HR documents — Clean up salary spreadsheets and org charts
  • Test data — Create realistic but anonymized datasets

Supported Formats

  • .xlsx — Microsoft Excel 2007 and later
  • .ods — OpenDocument Spreadsheet (LibreOffice, OpenOffice)

Note: Legacy .xls format is not supported. Please save as .xlsx first.

Output Options

  • Download XLSX/ODS — Anonymized spreadsheet in original format
  • Export as CSV — Plain text for further processing
  • Export mapping — Original→replacement dictionary
