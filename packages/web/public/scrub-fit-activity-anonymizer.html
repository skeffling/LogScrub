FIT File Anonymizer

Remove sensitive data from Garmin, Wahoo, and fitness device files

FIT (Flexible and Interoperable Data Transfer) files from sports watches, bike computers, and fitness devices contain detailed personal data including GPS tracks, heart rate, user profiles, and timestamps. LogScrub helps you anonymize this data before sharing activities or uploading to third-party services.

Privacy Risks in FIT Files

Data Type Privacy Risk
GPS Coordinates Reveals home address, workplace, and travel patterns
Timestamps Shows when you're away from home, daily routines
User Profile Contains personal details like name, weight, age, gender
Device Info Serial numbers can be used to identify your specific device
Heart Rate May reveal health conditions or fitness level

How It Works

LogScrub analyzes your FIT file and shows you exactly what sensitive data it contains. Since FIT is a binary format that's complex to rewrite, LogScrub exports the data to JSON format where it can be fully anonymized.

Export to JSON

FIT files are exported to JSON for anonymization. The resulting JSON contains all activity data in a human-readable format that can be used for analysis or converted to other formats like GPX.

Analysis Report

When you load a FIT file, LogScrub provides a detailed report including:

  • Number of GPS points recorded
  • GPS bounding box (geographic area covered)
  • Activity duration and distance
  • Data types present (heart rate, power, etc.)
  • User profile fields detected
  • Device manufacturer and model

  • Bug Reports — Report issues to fitness app developers without exposing personal data
  • Research — Contribute activity data to research projects anonymously
