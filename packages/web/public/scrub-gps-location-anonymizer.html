Why GPS Privacy Matters

GPS tracks can reveal your home address (where runs start/end), workplace location, daily routines, and travel patterns. Even "public" activity sharing can expose sensitive location data.

FIT File Processing

FIT files are Garmin's binary format. LogScrub converts them to GPX format for anonymization. The GPX output includes GPS track, elevation, timestamps, heart rate, cadence, power, and temperature data. You can then anonymize the GPX and download the result.

FIT to GPX Workflow

FIT file → Convert to GPX → Anonymize coordinates → Download GPX

Anonymization Methods

Coordinate Shifting

Offset all coordinates by a random amount. The track shape is preserved but moved to a different location.

Precision Reduction

Reduce GPS decimal precision. Full precision (6 decimals) = ~10cm accuracy. Reduced (3 decimals) = ~100m accuracy.

Start/End Trimming

Remove the first and last N points to hide where the activity started and ended (typically home/work).

Timestamp Shifting

Offset all timestamps to obscure when the activity occurred, hiding your daily schedule.

What Gets Detected

  • Track coordinates - Latitude/longitude in track points
  • Waypoints - Named locations (often home/work)
  • Timestamps - Date/time information revealing schedules
  • User profile - Name, weight, age (FIT files)
  • Device info - Serial numbers, manufacturer (FIT files)
  • Sensor data - Heart rate, power, cadence (preserved in export)

FIT File Data Preserved in GPX Export

When converting FIT to GPX, the following data is included:

Data Type GPX Element
