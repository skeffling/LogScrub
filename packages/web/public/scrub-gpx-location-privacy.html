GPX Anonymizer

Protect your location privacy in GPS track files

GPX files from fitness apps, GPS devices, and mapping software contain precise location data that can reveal where you live, work, and travel. LogScrub helps you anonymize this sensitive data before sharing routes, activities, or location logs.

📸 Screenshot placeholder: GPX file with coordinate anonymization options
🔒 Why GPS Privacy Matters

GPS tracks can reveal your home address (where runs start/end), workplace location, daily routines, and travel patterns. Even "public" activity sharing can expose sensitive location data.

Anonymization Methods

📍 Coordinate Shifting

Offset all coordinates by a random amount. The track shape is preserved but moved to a different location.

🎯 Precision Reduction

Reduce GPS decimal precision. Full precision (6 decimals) = ~10cm accuracy. Reduced (3 decimals) = ~100m accuracy.

✂️ Start/End Trimming

Remove the first and last N points to hide where the activity started and ended (typically home/work).

🗑️ Waypoint Removal

Remove named waypoints that may contain location names, addresses, or personal annotations.

What Gets Detected

  • Track coordinates — Latitude/longitude in <trkpt> elements
  • Waypoints — Named locations in <wpt> elements
  • Route points — Planned routes in <rtept> elements
  • Metadata — Creator name, device info, activity names
  • Timestamps — Date/time information (can reveal schedules)
  • Embedded text — Descriptions, comments, annotations

Text Content Anonymization

GPX files often contain text that reveals personal information:

Metadata and Descriptions
Before
<metadata>
  <name>Morning run from home</name>
  <author>

After
<metadata>
  <name>Morning run from home</name>
  <author>
    <name>[NAME-1]</name>
    <email>[EMAIL-1]</email>
  </author>
  <desc>5K loop around Oak Park</desc>
</metadata>
